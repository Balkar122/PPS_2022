## Program 19: Tutorial Program #4
```
#include <stdio.h>
int main ()
{
float a;
double b;
puts("Enter age");
scanf("%f" ,&a);
puts("Child age");
scanf("%lf" ,&b);
printf("Your age = %f\n" ,a);
printf("Child age = %f\n" ,b);
return 0;
}
```

40
