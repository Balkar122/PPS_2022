## Program 6: Write a program to show bitwise operator
```
#include<stdio.h>
int main ()
{
int a,b;
printf("Enter two number:");
scanf("%d",&a);
scanf("%d",&b);
printf("A & B = %d\n" ,a&b);
printf("A | B = %d\n" ,a|b);
printf("~A = %d\n" ,~a);
return 0;
}
```
## Output :
```
Enter two number:5
10
A & B = 0
A | B = 15
~A = -6

```
