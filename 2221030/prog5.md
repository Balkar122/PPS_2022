## Program 5: Write a program to show use of if statements
```c
#include<stdio.h>
int main ()
{
int x;
printf("Enter a number : ");
scanf("%d",&x);
if(x>50){
printf("number is greater than 50");
}
else{
printf("number is less than 50");
}
}

```
## Output :
```
Enter a number : 40
number is less than 5
```
